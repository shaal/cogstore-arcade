How to install on your Cognitum Seed
  1. Add the store by URL. In your Seed dashboard → Stores → Add store, paste https://arcade.shaal.dev/app-registry.json.
  2. Confirm the fingerprint (trust-on-first-use). The Seed will show a key fingerprint — check it matches …, then pin it.
  3. Install a game. It now appears in your store list; install it as shaal-arcade/<cog> and launch from the dashboard.

Requires a Seed runtime with custom-store support (configurable store + https:// fetch + signature verify). Everything the Seed checks is in the signed store.json / app-registry.json.
